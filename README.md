# Taller-Grupal-de-POO
# Sistema de Gestión de Biblioteca - Taller POO

Este repositorio contiene el desarrollo de un sistema de gestión para una Biblioteca Local, diseñado bajo el paradigma de Programación Orientada a Objetos (POO). El sistema permite administrar el catálogo de libros, usuarios, préstamos y devoluciones con validaciones automáticas de disponibilidad y multas.

---

## 🚀 Funcionalidades Principales
* **Gestión de Catálogo
* **Préstamos Inteligentes
* **Control de Disponibilidad
* **Reportes Ejecutivos
